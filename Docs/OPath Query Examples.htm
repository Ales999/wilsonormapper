Summary

This document contains a large number of example OPath queries written for SQL Server 2000's version of the Northwind database.  Each example shows the constructor portion of an OPathQuery.  You can click on any of the OPathQuery code fragments to show the SELECT statement that is generated when that OPathQuery is executed.  Only the T-SQL statements for SQL Server are shown in this document.  The OPath query engine will produce slightly different SELECT statements when using Microsoft Access or Oracle and can be configured to compatible statements for almost any other type of database.

The generic OPathQuery<T> class (available when using .NET Framework 2.0) was used in these examples to help streamline the code and highlight the object type being queried and the OPath expression.  For more details about creating and executing queries, see OPath Query API.  For details about the OPath language, see OPath Language.

NOTE   The examples in this document were extracted from test cases used to validate the OPath engine.  Many of the examples demonstrate important features of the OPath language but produce database queries that have no practical meaning.
